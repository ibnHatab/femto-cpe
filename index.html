Femto Access Point (Customer Premises Equipment)

A Femto Access Point (FAP), or “Femtocell” is a small-scale cellular base station, typically designed for use in a home or small business. FAPs will be remotely managed from mobile network operators using TR-069/CWMP. 'femto-cpe implements CWMP agent, configuration database and FAP OAM entity.

Dependencies

[1] TR-069 Amendment 4, CPE WAN Management Protocol, Broadband Forum, 2010 [6] TR-181 Issue 2 Amendment 3, Device Data Model for TR-069 (Device:2), Broadband Forum 2011 [7] TR-196 Issue: 2 Femto Access Point Service Data Model Broadband Forum 2011
